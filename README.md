# 股票 DCF 估值模型 · A股 / 港股 / 美股

一个**单文件、纯前端**的交互式 DCF 估值计算器。双击 `index.html` 即可用浏览器打开，无需后端、无需安装。

在线体验：<!-- DEPLOY_URL -->（部署后填入 GitHub Pages 链接）

## 功能

- **两种估值模式**
  - 营收驱动 FCFF（A 股，自动抓取年报营收）
  - 净利驱动 FCFE（A / 港 / 美 通用，市值 ÷ PE(TTM) 反推净利）
- **多市场 + 货币自洽**：A 股人民币、港股港元、美股美元，结果即该币种每股内在价值
- **实时抓取**：东方财富为主、腾讯行情为备用源（JSONP 绕过 CORS，`file://` 直接可用）；自动解析 A 股 6 位 / 港股 5 位 / 美股字母代码
- **输出**：每股内在价值、潜在空间、低估/合理/高估判定、企业/股权价值拆解、10 年现金流明细、WACC × 永续增速 5×5 敏感性矩阵
- **反向 DCF**：现价隐含折现率、现价隐含增长率（一眼看出"市场在赌多高增长/要多高回报"）
- **安全边际**：建议买入价 = 内在价值 ×(1−MOS)，并提示"风险双罚"
- **三视角投资评判**：巴菲特 / 芒格 / 段永平的通用判据框架 + 按当前股票实时量化初筛（ROE = PB/PE 等）
- **防护**：亏损/PE≤0 时"无法估值"拦截、净现金=0 醒目警示、跨公司数据污染防护

## 数据来源

实时行情与 A 股年报数据来自东方财富 / 腾讯财经公开接口。**仅供分析与教学，不构成投资建议。** DCF 对 WACC 与永续增速高度敏感，请结合敏感性矩阵综合判断。

## 注意

- 折旧、资本开支、净现金（标 ✱）接口无法稳定获取，请按目标公司最新年报手填。
- 数据接口为国内财经站点，海外网络可能访问受限。
